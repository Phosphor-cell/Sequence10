STORAGE ARCHITECTURE: HYBRID (Best & Free)
================================================================================

LOCAL SQLITE (Every device):
  ├─ Player state (fast, instant)
  ├─ Summoned heroes (fast)
  ├─ Loot inventory (fast)
  ├─ Narrative chapters (fast)
  └─ sync_queue (tracks offline actions)

NEON POSTGRESQL (Cloud):
  ├─ Single source of truth
  ├─ Battle validation (cheating protection)
  ├─ LLM cache (avoid regenerating)
  ├─ Cross-device sync
  └─ Daily backups

SYNC DAEMON (Background Thread):
  ├─ Every 5 seconds: Check network
  ├─ If online: Upload queued actions to Neon
  ├─ If offline: Queue locally, retry later
  ├─ Merge conflicts: Server data wins
  └─ No lag, always works

COST: $0/month (SQLite free, Neon free tier)
